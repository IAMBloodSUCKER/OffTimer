## Что это

OffTimer — небольшое приложение, которое выполнит выбранное действие в заданное время и предупредит за минуту до события.

**Доступные действия:**

| Действие | Описание |
|----------|----------|
| Выключение | Завершение работы компьютера |
| Перезагрузка | Перезапуск системы |
| Гибернация | Переход в режим гибернации |
| Сон | Переход в спящий режим |

## Установка

1. Откройте страницу [**релизов**](https://github.com/IAMBloodSUCKER/OffTimer/releases/latest)
2. Скачайте **`OffTimer-Windows.zip`**
3. Распакуйте архив в любую папку
4. Запустите **`OffTimer.exe`**

Установщик не нужен — распаковка занимает несколько секунд. Java уже внутри программы.

## Удаление

**Портативная версия (zip):** закройте OffTimer и удалите папку `OffTimer`.

**Старая версия с установщиком (до v1.0.8):** сначала закройте OffTimer через трей → **Выход**, затем:

1. **Параметры Windows** → **Приложения** → **OffTimer** → **Удалить**
2. В диалоге с «??????» выберите **Do not close applications** → **OK**

Или запустите скрипт от имени администратора:

```powershell
powershell -ExecutionPolicy Bypass -File scripts/uninstall.ps1
```

После удаления скачайте актуальный **`OffTimer-Windows.zip`** — он не использует установщик Windows.

## Как пользоваться

1. Выберите **действие** в выпадающем списке
2. Укажите время одним из способов:
   - **через** — количество часов и минут от текущего момента
   - **в точное время** — время в формате `ЧЧ:ММ` (24-часовой)
3. Нажмите **Запустить**
4. За 60 секунд до события появится уведомление — его можно отменить
5. Крестик окна сворачивает приложение в трей; двойной щелчок по иконке возвращает окно

Настройки сохраняются автоматически в файл `offtimer-settings.json` рядом с программой.

## Сборка из исходников

Требуется [JDK 17+](https://adoptium.net/) и [Maven](https://maven.apache.org/):

```bash
mvn package
java -jar target/offtimer.jar
```
