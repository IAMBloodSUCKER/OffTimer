# OffTimer
Минималистичное десктопное приложение для планирования автоматического выключения, перезагрузки, гибернации или перевода компьютера в режим сна. Установите таймер «через N минут» или укажите точное время — и приложение само выполнит действие в нужный момент, предупредив вас за 60 секунд.
